# Анализ корпуса стихотворений А. Галича

## Описание
В этом корпусе содержатся стихотворения поэта А. А. Галича, написанные с 1961 по 1977 год. В рамках проекта релизуется подсчет следующих текстовых характеристик: количеств слов (в том числе уникальных), число строк, средняя длина слова, TTR (показатель лексического разнообразия текста) и др.        

## Структура Проекта

Проект включает в себя корпус стихотворений, записанных в формате текстовых файлов, папку "data", в которой находится файл "metadata.csv" (в нем содержится информация о названиях стихотворений, их авторе и годах написания), папку "results", в которой находятся файлы со статистикой и результатами анализа корпуса ("statistics.csv" и "report.txt"), а также файлы с реализованными в рамках работы над анализом корпуса функциями. Более подробная структура проекта представлена ниже: 

your_project_name/
├── corpus/              # Папка с текстовыми файлами
│   ├── text_1.txt
│   ├── text_2.txt
│   └── ...
├── data/                # Структурированные данные
│   └── metadata.csv
├── results/             # Результаты анализа (создаётся автоматически)
│   ├── statistics.csv
│   └── report.txt
├── main.py              # Главный файл с точкой входа
├── text_utils.py        # Функции анализа текста
├── file_utils.py        # Функции работы с файлами
├── README.md            # Документация проекта
└── PROJECT_REQUIREMENTS.md  

## Как Запустить
```bash
python main.py
```
